## HTML Syntax and Structure

HTML is a Markup language.
We use tags <> to mark content up.
Example: 
        <h1> This is a header tag </h1>
        <p> This is a paragraph tag </p>
        <h2 id="main-content"> Let's use an id </h2>
        <p class="footer-link"> Let's use a class </p>
        <span style="color: blue;"> Let's use inline styling </span>

### Syntax Summary

HTML uses tags <> </> to present content
There are so many different tags
Tags can contain attributes like <p class="myclass">
Tags can be nested in other tags
Tags are normally indented when nested for readability
